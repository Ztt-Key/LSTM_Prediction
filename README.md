# 🎵 中文歌词预测系统

基于深度学习的中文歌词生成AI助手，输入一句歌词，AI为您创作下一句！

## ✨ 特色功能

- 🎯 **多样化预测**: 每次生成3个不同风格的歌词续句
- 🎨 **多种风格**: 支持古诗词、现代流行、民族风格等
- 🌐 **Web界面**: 现代化的网页交互界面
- 💻 **命令行模式**: 适合开发者的终端界面
- 🔄 **智能防重复**: 避免生成相同的预测结果
- ⚡ **快速响应**: 优化的模型架构，预测速度快

## 🚀 快速开始

### 环境要求

- Python 3.11+ (推荐)
- 系统支持: Windows/Linux/MacOS

### 安装依赖

```bash
pip install -r requirements.txt
```

#### 🎯 方式一：优化版预测器 (推荐)

- ✅ **响应速度快**: 轻量级设计，秒级响应

#### 🎯 方式二：完整系统启动（正在改进）

提供多种模式选择：

## ⚙️ 技术特点

### 🧠 AI模型架构
- **LSTM神经网络**: 处理序列数据的经典架构
- **注意力机制**: 提升上下文理解能力
- **多头注意力**: 从多个角度理解输入
- **梯度裁剪**: 防止梯度爆炸，训练更稳定

### 🎨 预测策略
- **多温度采样**: 不同温度产生不同风格
- **Top-k采样**: 在最可能的词中随机选择
- **模板生成**: 基于预定义模板的备用策略
- **规则生成**: 简单规则作为最终备选

### 📊 数据处理
- **智能分词**: 使用jieba进行中文分词
- **数据清洗**: 自动过滤和规范化文本
- **序列生成**: 多种长度的滑动窗口训练
- **词汇优化**: 智能构建高质量词汇表
